    Used to define the transitions out of a state.

    Used to define the transitions out of a state. The transitions are modeled as a single partial function taking an event as argument, and returning a target, a pair (s,code) consisting of the target state s and the code to execute when firing that transition. The transition function is only defined for those events for which it can fire, hence a partial function.
