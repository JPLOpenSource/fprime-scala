  6.  def bddFromBinding(binding: Option[Binding]): BDD

    Turns an optional binding from variable names to values (an assignment) into a BDD.

    Turns an optional binding from variable names to values (an assignment) into a BDD. This is achieved by computing the BDD for each variable/value pair and the AND-ing these BDDs together. The function is called when an event pattern has matched an incoming event in the state.

    binding

    the binding to convert into a BDD.

    returns

    the BDD resulting from and-ing the BDDs for each variable binding in binding.
