BDDGenerator

class BDDGenerator extends AnyRef

An object of this class represents all the variables in a formula. It contains a mapping from variable names (strings) to objects of class Variable, each of which contains the hashmap from values of the corresponding variable to BDDs.

  14.  def getFreeBDDOf(varName: String, formula: BDD): BDD

    Collects the garbage for a variable in a sub-formula.

    Collects the garbage for a variable in a sub-formula. This is done using the formula:

    forall y0,...,z0,... . (F[1/x0,...,1/xn] <-> F)

    where x is the variable, and x0,x1,...,xn are the bit positions for that variable, and y0,...,z0,... are the bit positions for all other variables y, z, .... The formula defines a BDD which accepts values v for x (in F) such that F[v/x] is identical to F[1/x0,...,1/xn]. Those are the values that are no longer needed, hence can be garbage collected. Recall that 111..1 represents all values not yet seen.

    varName

    the name of the variable being garbage collected (x in the above example).

    formula

    the formula being garbage collected over (F in the above example).

    returns

    the free assignments.
