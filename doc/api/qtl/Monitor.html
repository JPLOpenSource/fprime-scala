Monitor

abstract class Monitor extends AnyRef

The generic Monitor class. A specialized monitor for a set of properties must extend this class. It contains the BDD generator (which generates the association between values and BDDs), the state (which contains the current event), and the list of user provided formulas. In addition it provides a set of options that can be set by the user.
